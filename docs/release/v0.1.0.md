## OOPforge v0.1.0 — Initial release

**Forge small. Compose forever.**

Portable OOP/DDD methodology pack for AI coding agents (Claude Code, Codex CLI, Cursor). Small skills, explicit workflow stages, human checkpoints, and runnable proof — not mega-prompts.

### Highlights

- **Workflow** — Discovery → Design → Delivery Plan → Skeleton → Implement → Test (+ Refactor)
- **Skills** — OOP/DDD patterns, Java Spring hexagonal & Python FastAPI layouts (≤200 lines per skill)
- **Craft** — single `/oopforge:craft` entry point
- **Examples** — runnable `order-java` and `order-python` (place-order)
- **Onboarding** — [library loan walkthrough](docs/guides/library-loan/README.md) (EN/KO/JA/ZH)
- **Install** — symlink-based setup for Claude Code & Codex CLI
- **CI** — skill lint + isolated HOME smoke test

### Quick install

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/LooSung/oopforge/main/scripts/setup/bootstrap.sh)"
```

From `~/.oopforge`:

```bash
./scripts/setup/install.sh
./scripts/setup/doctor.sh
./scripts/setup/install.sh update   # after git pull
```
