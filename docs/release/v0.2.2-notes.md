## OOPforge v0.2.2 — Runnable proofs for every stack

**Forge small. Compose forever.**

Four skeleton stacks, four runnable examples — same `place-order` flow. Python is FastAPI-only (layered + clean).

### Highlights

- **Examples** — `order-java-layered` (Spring 3-tier), `order-python-layered` (FastAPI 3-tier), plus existing hexagonal proofs
- **Index** — [examples/README.md](examples/README.md) maps stack command ↔ folder
- **4 stacks** — removed `python-flask-layered`; Java layered/hexagonal + Python FastAPI layered/clean
- **Tests** — each layered example: domain tests + OpenAPI smoke

### Quick verify

```bash
cd examples/order-java-layered && ./gradlew test
cd examples/order-python-layered && pip install -e ".[dev]" && pytest
```

### Install

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/LooSung/oopforge/main/scripts/setup/bootstrap.sh)"
```

From `~/.oopforge`:

```bash
./scripts/setup/install.sh
./scripts/setup/doctor.sh
./scripts/setup/install.sh update   # after git pull
```
