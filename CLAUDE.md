# OOPforge — Claude Code Instructions

Read and follow @AGENTS.md.

When working on this repository:

- Use `/oopforge:discovery`, `/oopforge:design`, `/oopforge:skeleton`, and `/oopforge:implement` as the canonical workflow commands when available.
- Use `@ddd-architect` for DDD modeling work when available.
- Keep Claude-specific packaging in `.claude-plugin/`, `agents/`, and `commands/`.
- Do not bypass the Discovery → Design → Skeleton → Implement workflow for new domains or large features.
